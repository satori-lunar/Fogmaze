Code Flow
=========

User Action â†’ ContentView â†’ GameViewModel â†’ Model â†’ View Update

1. User swipes on screen
2. ContentView.handleSwipe() detects direction
3. GameViewModel.movePlayer() validates and updates position
4. Maze.swift data checked for walls/exit
5. GameViewModel updates @Published properties
6. MazeView automatically re-renders
7. Smooth animations play
